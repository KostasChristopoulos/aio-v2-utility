# AIO v2 - Multipurpose CSV Tool

A multi-purpose Python application with a GUI to perform various CSV operations.

## Features
- **CSV Batch Splitter**: Split large CSV files into smaller batches.
- **CSV Column Dropper**: Remove specific columns from a CSV file.
- **Array to String Converter**: Convert array-like strings to pipe-delimited strings across all columns.
- **CSV Concatenator**: Merge multiple CSV files from a folder.
- **Drag & Drop Support**: Easily drop files or folders into the app.
- **Activity Log**: Real-time tracking of all operations.
- **Persistent Memory**: Remembers your last used settings.

## Installation
1. Clone the repository.
2. Install dependencies:
   ```bash
   pip install customtkinter pandas Pillow tkinterdnd2
   ```

## Usage
Run the main script:
```bash
python main.py
```
